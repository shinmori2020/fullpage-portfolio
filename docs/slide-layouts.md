# スライドレイアウト定義書

参考画像: `docs/reference/pages.jpg`

このファイルは参考画像の各スライド（1〜27）のレイアウトを文章で定義したものである。
「スライドNを実装して」と指示された場合、該当番号のレイアウト定義に従ってHTMLパネルを作成すること。

## 実装ルール

- 各スライドは `<section class="panel slide-N">` として実装する（Nはスライド番号）
- アニメーション対象の要素には `.anim-hidden` を付与する
- 画像はプレースホルダー（背景色 + アイコンまたはダミーテキスト）で仮実装する
- 配色・フォントは `CLAUDE.md` のデザイン仕様に従う
- テキストは仮テキストで実装し、後から差し替える

### 共通カラー参照

```
メインブルー背景:  var(--accent)  #2c5fcc
ライトブルー背景:  var(--accent-light)  #e8eefb
白背景:            var(--bg-white)  #ffffff
グレー背景:        var(--bg-light)  #f6f7f9
濃いテキスト:      var(--text-dark)  #1a1e2c
本文テキスト:      var(--text-body)  #4a4f63
薄いテキスト:      var(--text-muted)  #8b90a0
ボーダー:          var(--border)  #e2e4ea
```

## スライド1: カバー / Hero

- **背景**: 全画面ブルー（`--accent`）
- **レイアウト**: 中央下寄せ
- **要素**:
  - 背景に建物（ビル）の写真をオーバーレイ（暗めフィルター付き）
  - 左下に大きなタイトル「REPORT」（白、極太、英語、フォントサイズ特大）

- **装飾**: 右上に斜めの幾何学的な線（白い斜線が複数本、装飾的に配置）
- **印象**: 力強いカバースライド、企業プレゼンの表紙
